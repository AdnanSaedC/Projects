# 05 - Quiz App

An interactive **Quiz App** built with vanilla JavaScript. It presents multiple-choice questions one at a time, tracks the score, and shows the final result at the end.

## Features

- Start / Restart quiz flow
- Multiple-choice questions rendered dynamically
- Score tracking
- Final score display at the end
- Next question button

## Sample Questions

- General knowledge (capitals, science, literature)
- Easily extendable by adding objects to the `questions` array in `scriptByMe.js`

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. No build step or dependencies required.

## File Structure

```
05_Quiz/
├── index.html        # Main HTML structure
├── scriptByMe.js     # Core logic (questions array, score tracking, rendering)
└── styles.css        # Styling
```
