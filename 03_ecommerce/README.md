# 03 - Simple E-Commerce

A simple **E-Commerce** web app built with vanilla JavaScript. It displays a product listing and a shopping cart with quantity management, all persisted using `localStorage`.

## Features

- Dynamically rendered product list
- Add items to the shopping cart
- Quantity increment for duplicate items
- Cart total calculated automatically
- Cart persisted with `localStorage`
- Checkout button

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. No build step or dependencies required.

## File Structure

```
03_ecommerce/
├── index.html    # Main HTML structure
├── script.js     # Core logic (product rendering, cart management, localStorage)
└── styles.css    # Styling
```
