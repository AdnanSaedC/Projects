# 04 - Expense Tracker

A simple **Expense Tracker** web app built with vanilla JavaScript. It lets you log expenses with names and amounts, displays a live total, and persists data using `localStorage`.

## Features

- Add expenses with a name and amount
- Displays a running total of all expenses
- List of all logged expenses
- Data persisted with `localStorage`
- Form validation (required fields)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. No build step or dependencies required.

## File Structure

```
04_expense_tracker/
├── index.html        # Main HTML structure
├── scriptByMe.js     # Core logic (add/delete expenses, total, localStorage)

```
