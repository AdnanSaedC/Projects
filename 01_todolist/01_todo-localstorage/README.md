# 01 - To-Do List with Local Storage

A simple **Dark Mode To-Do List** web app built with vanilla JavaScript. Tasks are persisted in the browser's `localStorage`, so they survive page refreshes.

## Features

- Add new tasks via an input field
- Mark tasks as completed (strikethrough toggle)
- Delete individual tasks
- Data persistence using `localStorage`
- Dark mode UI

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. No build step or dependencies required.

## File Structure

```
01_todo-localstorage/
├── index.html        # Main HTML structure
├── scriptByMe.js     # Core logic (localStorage, add/delete/toggle tasks)
├── script.js         # Reference/tutorial script
└── styles.css        # Dark mode styling
```
